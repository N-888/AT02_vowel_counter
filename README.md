# 🔤 Подсчёт гласных с PyTest

Этот проект демонстрирует, как с помощью **PyTest** можно легко и элегантно тестировать простую функцию подсчёта гласных букв в строке.

## ✨ Особенности
- Функция `count_vowels` учитывает как строчные, так и прописные буквы.
- Тесты покрывают:
  - строки из одних гласных,
  - строки без гласных,
  - смешанные строки,
  - граничные случаи (пустая строка, пробелы, цифры).
- Использование **параметризованных тестов** позволяет сократить дублирование кода.
- Чистая структура проекта: один файл с функцией, один – с тестами.

## 🚀 Запуск тестов
1. Убедись, что установлен pytest:  
   `pip install pytest`
2. В терминале выполни:  
   `pytest test.py -v`
3. Если все тесты пройдены, увидишь зелёные галочки.

## 📊 Пример использования
```python
from main import count_vowels

print(count_vowels("Hello, World!"))   # 3
print(count_vowels("Python"))          # 1
